# Mqrtin's Utility (Forge 1.8.9)

Setup minimal Forge 1.8.9 avec Loom/Architectury.

## Prerequis

- Java 8 (JDK)
- Gradle wrapper du projet

## Commandes utiles (PowerShell)

```powershell
.\gradlew.bat tasks
.\gradlew.bat build
.\gradlew.bat runClient
```

## Fichiers principaux

- `build.gradle.kts`
- `settings.gradle.kts`
- `gradle.properties`
- `src/main/java/fr/mqrtin/main.java`
- `src/main/resources/mcmod.info`
